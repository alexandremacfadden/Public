Insertion Sort

Sort A[1..2], then A[1..3], then A[1..4], and so on.

Insertion-Sort(A, n) for j ← 2 to n key ← A[j] i ← j − 1 while i > 0 and A[i] > key do A[i+1] ← A[i] i ← i − 1 A[i+1] ← key return A

Running time: Θ(n²) worst case, Θ(n) best case

Sorting Problem

Given an array A[1..n], design an algorithm for re-arranging its elements in ascending order:

A[1] ≤ A[2] ≤ ... ≤ A[n]

The problem is very important! Most software systems use sorting.
